# 🌤️ Weather Dashboard

A modern, responsive **Weather Dashboard** built using **HTML, CSS, and JavaScript**. This application allows users to search for any city and instantly view real-time weather information through the **OpenWeather API**. The project demonstrates API integration, asynchronous JavaScript, responsive UI design, and dynamic DOM manipulation.

🌐 **Live Demo:** https://weather-app-by-hasaan.vercel.app/

📂 **GitHub Repository:** https://github.com/HasaanDev-web/weather-app

---

## 📌 Overview

The Weather Dashboard provides current weather conditions for cities around the world with a clean, user-friendly interface. It uses the **OpenWeather API** to fetch live weather data and displays useful information such as temperature, humidity, wind speed, and weather conditions.

This project showcases my frontend development skills, especially working with APIs, asynchronous JavaScript, and responsive web design.

---

## ✨ Features

- 🌍 Search weather by city name
- 🌡️ Real-time temperature updates
- 💧 Humidity information
- 💨 Wind speed display
- ☁️ Dynamic weather conditions & icons
- 📍 Default weather on page load
- 🔄 Loading animation while fetching data
- ❌ Error handling for invalid city names
- 📱 Fully Responsive Design
- ⚡ Fast and lightweight interface

- 🌙 Smooth animations and transitions

## 🔑 API Setup

This project uses the **OpenWeather API** to fetch live weather data.

1. Create a free account at:
   https://openweathermap.org/api

2. Generate your API key.

3. Replace the API key inside `script.js`:

```javascript
const API_KEY = "YOUR_API_KEY";
```
